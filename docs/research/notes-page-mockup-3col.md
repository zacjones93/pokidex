## Grid Structure

```tsx
// home-outlet.tsx (parent)
<div className="grid lg:grid-cols-[300px_1fr_1fr] gap-6">
  {/* Column 1: List (300px fixed) */}
  <div>...</div>

  {/* Column 2 & 3: Filled by $id route outlet */}
  <Outlet />
</div>

// home-outlet.$id.tsx returns Fragment with 2 divs
<>
  {/* Column 2: Detail */}
  <div>...</div>

  {/* Column 3: Notes outlet */}
  <div>
    <Outlet /> {/* home-outlet.$id.notes.tsx */}
  </div>
</>
```

## Column Breakdown

### Column 1: Pokemon List (300px fixed)
- **Header**: "Pokédex" + subtitle
- **Grid**: 2 columns x 10 rows = 20 Pokemon
- **Cards**: Compact buttons with:
  - Pokemon ID (#001)
  - Pokemon name (abbreviated if long)
- **Pagination**: Centered, compact
- **Page info**: "Page X of Y (N total)"

### Column 2: Pokemon Detail (1fr - flexible)
- **Pokemon Card**: Full variant
  - Large sprite
  - ID, Name, Type badges
  - Height, Weight
  - Abilities
  - Stats with bars
- **Evolution Chain**: Card below
  - Visual evolution path
  - Evolution methods
  - Current stage

### Column 3: Notes (1fr - flexible)
- **Add Note Form**:
  - Textarea (4 rows)
  - "Add Note" button
- **Notes List** (scrollable):
  - Note content
  - Timestamp
  - Delete button [×]
  - Newest first

## Responsive Behavior

### Desktop (lg+)
```
[List 300px] [Detail 1fr] [Notes 1fr]
```
- 3 columns side-by-side
- All columns scrollable independently

### Tablet (md to lg)
```
[List]
[Detail]
[Notes]
```
- Stacks vertically
- Full width each

### Mobile (<md)
```
[List]
[Detail]
[Notes]
```
- Single column
- Condensed list (1 column grid)

## Outlet Nesting

```
/home-outlet
└─ home-outlet.tsx (3-col grid: list + outlet)
   └─ /home-outlet/:id
      └─ home-outlet.$id.tsx (Fragment: detail + outlet)
         └─ /home-outlet/:id (index route)
            └─ home-outlet.$id.notes.tsx (notes with defer/Suspense)
```

## Empty States

1. **No Pokemon Selected** (`/home-outlet`):
   - Columns 2 & 3 show: "← Select a Pokémon to view details"

2. **Pokemon Selected, No Notes** (`/home-outlet/25`):
   - Column 2: Pokemon detail renders
   - Column 3: "No notes yet. Add your first note about Pikachu!"

## Styling Notes

- All columns have max-height with overflow-y-auto
- Cards use consistent shadows and borders
- Gap between columns: `gap-6` (1.5rem / 24px)
- List column: Fixed 300px width on desktop
- Detail/Notes columns: Equal flexible width (`1fr` each)
